Settings for .NET

To make a new trace connection for an IIS-based .NET program, modify this command and run as an Administrator: CodePulse.DotNet.Tracer.exe -IIS "-TargetDir:<targetdir>" "-IISAppPoolIdentity:<domain\username>"
To make a new trace connection for an executable .NET program, modify this command and run: CodePulse.DotNet.Tracer.exe "-Target:<target application>" -SendVisitPointsTimerInterval:<milliseconds>
